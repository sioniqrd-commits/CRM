📊 Google Sheets Sync

Push all CRM data to your Google Spreadsheet in one click

⚙ Setup — Web App URL
One-time setup (5 minutes):
1. Open script.google.com → New Project
2. Paste the SioniqCRM_GoogleAppsScript_v2.js code into the editor
3. Replace YOUR_SPREADSHEET_ID_HERE with your Google Sheet ID
4. Click Deploy → New Deployment → Web App
   • Execute as: Me  |  Who has access: Anyone
5. Copy the Web App URL and paste it below
💡 Already deployed? If you deployed the old script, go to Deploy → Manage Deployments → Edit → New Version, paste the new v2 code, then redeploy. The URL stays the same.
